# ChangelogSwift
Manage CHANGELOG.md with Swift


# Commands
- changelog.swift new-version 1.2.3 

# Notes
- the changelog need to be called CHANGELOG.md
- CHANGELOG.md need to be UTF8 encoded 

# CHANGELOG.md example
```
# Change Log

## [Unreleased]

### added
- New app icon

## [0.1.0] - 2015-12-21

### changed
- UI tweaks

### added
- Swipe cells to edit

### fixed
- Freeze when app enters in foreground
```
